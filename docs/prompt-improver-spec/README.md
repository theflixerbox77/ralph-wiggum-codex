# Prompt Improver Workspace

This directory stores repo-backed outputs created by `ralph-prompt-generator` during its staged prompt-improvement workflow.

## Layout

- `artifacts/implementation_plan.md`
- `artifacts/task.md`
- `artifacts/walkthrough.md`
- `final-prompts/<prompt-name>-draft.md`
- `final-prompts/<prompt-name>.md`

The draft prompt exists only during the workflow and should be deleted after the final prompt is created.
